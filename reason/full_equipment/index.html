選ばれる理由02 フル装備の家

フル装備の家

泉北ホームの「フル装備の家」PREMIUMパッケージは、生活に絶対必要なものが標準で付いているオールインワン住宅です。他社ならオプションや追加オーダーになるような高い住宅設備・構造が標準で付いています。しかも、それぞれの装備はどれも女性目線で選んだ高機能なものばかり。お引渡ししたその日から「快適」に暮らせます。もちろん、お客様のこだわりをさらに追加する事ことも可能です。

フル装備の設備

オプションなしでも大満足。
それが、私たちの「標準装備」です。

注文住宅では、お引渡し時に備え付けられているシステムキッチンやクローゼット、外壁材などの住宅設備等を「標準仕様」といいます。
「標準」の基準グレードは各社さまざまですが、一般的にはオプションでのグレードアップ・追加を見込んでか、標準仕様は「やや物足りない」グレードに抑えられているケースが多々あります。納得できる装備にしようとすると契約後にオプションの追加費用が数百万円かかってしまう、ということも少なくありません。
泉北ホームが目指すのは、オプションがなくとも「標準」で快適かつ大満足の家づくり。自社一貫施工だからこそ、お客様にとって無駄な費用（中間マージン）を徹底的に排除可能となり、また年間施工棟数が多く、大量発注できるため設備や建材のメーカー仕入れ価格を下げられるからこそ、「フル装備の家」を実現できています。

「暮らしがラクになる」を基準に大手設備・建材メーカーを採用。フル装備の家 PREMIUM PACKAGE 設備仕様

フル装備の構造

耐震性や断熱性・気密性にもこだわり。
万一の災害にも家族を守る構造。

外観や設備を充実させた「フル装備の設備」と対を成すのが、「フル装備の構造」です。
「設備」が「毎日」の暮らしやすさを助けてくれるものなら、「構造」は「万一」の災害にも家族を守ってくれるもの。台風や地震などの災害に見舞われたときも安全・安心な住まいをつくることは、私たち泉北ホームのハウスメーカーとしての使命だと考えています。在来工法や鉄骨造以上に、阪神淡路大震災での全壊・半壊ゼロの実績を持つ「2×4工法」に注力するのは、そのためです。

また、外壁材や断熱材そして窓についても高性能・高品質なものを「標準仕様」としておくことで、日々の光熱費を削減することはもちろん、将来的なメンテナンス費用を抑えることが可能に。総合的に、お客様にとってメリットの大きい家づくりを考えています。

建築基準法ではなく、長期優良住宅を最低基準に。フル装備の家 PREMIUM PACKAGE 構造・性能
